Zum ausführen wird eine Installation von Python benötigt.
Programmiert und getestet wurde das Softwareprojekt mit Python 3.3.
Eine Installationsdatei für Windows des frei verfügbaren Python ist beigelegt.
Die vorgefertigten Testläufe lassen sich mit start.bat ausführen.
Ein eigener Testlauf wird durch den Aufruf:

Controler.py [Filename]

gestartet.

Mehr Informationen erhalten Sie durch den Aufruf von:

Controler.py help

Ein Log der Ausaben und Fehlermeldungen wird in die Datei log.txt geschrieben.
